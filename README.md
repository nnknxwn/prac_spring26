# Метод продолжения по параметру

Решение краевых задач для обыкновенных дифференциальных уравнений методом продолжения по параметру. Проект включает численную реализацию метода и графический интерфейс.

## Стек

- **SymPy** — символьные вычисления, якобианы
- **NumPy** — работа с массивами
- **SciPy** — численное решение ОДУ (`solve_ivp`)
- **Matplotlib** — визуализация решений
- **Tkinter** — графический интерфейс

## Формат ввода

**Уравнения** — правые части системы ОДУ, переменные `x1, x2, ..., xn`, время `t`:
```
x2
-x1
```

**Краевые условия** — выражения равные нулю, `xa0..xan` — значения в точке `a`, `xb0..xbn` — в точке `b`:
```
xa0 - 0
xb0 - 0
```

**Начальное приближение** — значения через запятую: `0, 1`
