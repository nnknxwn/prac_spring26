# Метод продолжения по параметру

Решение краевых задач для обыкновенных дифференциальных уравнений методом продолжения по параметру. Проект включает численную реализацию метода и графический интерфейс.

## Стек

- **SymPy** — символьные вычисления, якобианы
- **NumPy** — работа с массивами
- **SciPy** — численное решение ОДУ (`solve_ivp`)
- **Matplotlib** — визуализация решений
- **Tkinter** — графический интерфейс

## Структура

```
prac_spring26/
├── main.py           — точка входа, запуск приложения
├── solver.py         — метод продолжения по параметру
├── io_manager.py     — сохранение и загрузка задач (JSON)
├── gui/
│   ├── app.py        — главное окно приложения
│   └── i18n.py       — тексты интерфейса (русский, английский)
├── examples/
│   └── two_body.py   — задача двух тел (пример из статьи)
├── tests/
│   └── test_solver.py
└── requirements.txt
```

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
