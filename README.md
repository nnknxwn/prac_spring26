# BVP Solver — Метод продолжения по параметру

Приложение для решения краевых задач систем обыкновенных дифференциальных уравнений методом продолжения по параметру (метод стрельбы с гомотопией).

Графический интерфейс на PyQt6 с поддержкой светлой/тёмной темы и двух языков (RU/EN).

---

## Математическая постановка

Дана система ОДУ:

$$\dot{x} = f(t, x), \quad t \in [a, b]$$

с краевыми условиями:

$$R(x(a), x(b)) = 0$$

где $x \in \mathbb{R}^n$, $R: \mathbb{R}^{2n} \to \mathbb{R}^n$.

### Метод решения

Задача решается в два уровня:

**Внутренняя задача** — для данного кандидата $p = x(t^*)$ вычисляется невязка $\Phi(p) = R(x(a; p), x(b; p))$ путём интегрирования расширенной системы (исходные переменные + матрица чувствительности).

**Внешняя задача** — поиск $p$ такого, что $\Phi(p) = 0$, методом продолжения по параметру (гомотопия):

$$\frac{dp}{d\mu} = -[\Phi'(p)]^{-1} \Phi_0, \quad p(0) = p_k, \quad \mu \in [0, 1]$$

Итерации повторяются до $\|\Phi(p)\| < \varepsilon$ или достижения максимального числа итераций.

## Формат ввода

### Уравнения ẋ = f(t, x)

Правые части системы ОДУ. Переменные: `x1`, `x2`, ..., `xn`, время: `t`.

```
x2
-x1
```

### Краевые условия R(x(a), x(b)) = 0

Выражения, которые должны обращаться в ноль. Обозначения:
- `xa0`, `xa1`, ... — значения $x_1(a)$, $x_2(a)$, ...
- `xb0`, `xb1`, ... — значения $x_1(b)$, $x_2(b)$, ...

```
xa0
xb0 - 1
```

### Начальное приближение p₀

Значения $x(t^*)$ — по одному в каждое поле.

---

## Пример

**Задача:** $x'' = 0$, $x(0) = 0$, $x(1) = 1$

**Решение:** $x(t) = t$

**Ввод (n = 2):**

| Поле | Значение |
|------|----------|
| x1' = | `x2` |
| x2' = | `0` |
| R1 = | `xa0` |
| R2 = | `xb0 - 1` |
| a | `0` |
| b | `1` |
| t* | `0` |
| p1₀ | `0` |
| p2₀ | `1` |

**Результат:** прямая $x_1 = t$, константа $x_2 = 1$.

## Методы интегрирования

| Метод | Тип | Когда использовать |
|-------|-----|-------------------|
| Рунге-Кутта 4(5) | Явный | Универсальный выбор для нежёстких задач |
| Рунге-Кутта 2(3) | Явный | Быстрый, но менее точный |
| Дорманд-Принс 8(5,3) | Явный | Высокая точность |
| Радо IIA | Неявный | Жёсткие системы |
| BDF | Неявный | Жёсткие системы с медленной динамикой |
| LSODA | Адаптивный | Автоматический выбор жёсткий/нежёсткий |
