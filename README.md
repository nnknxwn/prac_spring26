# Метод продолжения по параметру

Решение краевых задач для обыкновенных дифференциальных уравнений с помощью метода продолжения по параметру. Проект включает численную реализацию метода на Python и графический интерфейс для задания параметров задачи и визуализации результатов.

## Стек

- **SymPy** — символьные вычисления, формирование уравнений
- **NumPy** — работа с массивами и линейная алгебра
- **SciPy** — численное решение ОДУ и нелинейных систем
- **Matplotlib** — визуализация решений
- **Tkinter** — графический интерфейс
